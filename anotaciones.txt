Utilizo el entorno de ejecución Node.js y el bundler Vite (herramienta para combinar múltiples archivos como css y js en uno)

Instalar Homebrew
  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Instalar node.js
  brew install node

Verificar instalación
  node -v
  npm -v

Instalar Vite y crear package.json (si no lo tienes):
    npm init -y
    npm install vite --save-dev
Ejecutar docker-compose:
    docker-compose up --build
Para acceder a la aplicación:
    http://localhost:5173

Archivo package.json:

{
  "name": "frontend-project",
  "version": "1.0.0",
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "serve": "vite preview"
  },
  "devDependencies": {
    "vite": "^4.0.0"
  }
}

Certificados SSL/TLS con openssl:

Instalar openssl:
  apt-get -y install openssl

Crear el certificado:
  openssl req -x509 -nodes -days 365 -newkey rsa:2048 -keyout transcendence.key -out transcendence.crt

Añadir nginx a docker-compose. No hace falta Dockerfile porque podemos utilizar la imagen oficial de docker hub y configurar nginx.conf
